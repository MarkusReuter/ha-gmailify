# Privacy Policy

Gmailify is a self-hosted Home Assistant addon for personal use. It syncs emails from an IMAP mailbox to a personal Gmail account.

## Data Collection

This application does not collect, store, or share any personal data with third parties. All data remains on the user's own infrastructure.

## Gmail API Usage

The application uses the Gmail API solely to import emails into the authenticated user's own Gmail account. OAuth2 credentials are stored locally on the user's Home Assistant instance and are never transmitted to any external service.

## Contact

This is a personal project. For questions, open an issue on the GitHub repository.
